# Imágenes de la Aplicación

## Logo de Teamly

Para usar el logo en la aplicación, coloca tu imagen del logo en esta carpeta con el nombre:

**`logo.png`** o **`logo.jpg`**

### Instrucciones:

1. **Copia tu imagen del logo** a esta carpeta: `frontend/assets/images/`
2. **Nombra el archivo como:** `logo.png` (o `logo.jpg` si es JPG)
3. **Formatos soportados:** PNG, JPG, JPEG
4. **Tamaño recomendado:** 
   - Para mejor calidad: 512x512px o mayor
   - Mínimo: 256x256px

### Ubicación del archivo:
```
frontend/
  assets/
    images/
      logo.png  ← Coloca tu imagen aquí
```

Una vez que subas la imagen, el SplashScreen la usará automáticamente.
