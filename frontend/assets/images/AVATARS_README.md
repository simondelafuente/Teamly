# Avatares Predeterminados

Para que los avatares predeterminados funcionen correctamente, agrega 3 imágenes de avatar en esta carpeta con los siguientes nombres:

- `avatar1.png` (o `.jpg`)
- `avatar2.png` (o `.jpg`)
- `avatar3.png` (o `.jpg`)

## Instrucciones:

1. **Prepara tus imágenes de avatar:**
   - Formato recomendado: PNG o JPG
   - Tamaño recomendado: 512x512px o mayor (cuadradas)
   - Las imágenes deben ser cuadradas para verse mejor

2. **Coloca las imágenes en esta carpeta:**
   ```
   frontend/
     assets/
       images/
         avatar1.png  ← Coloca tu primera imagen aquí
         avatar2.png  ← Coloca tu segunda imagen aquí
         avatar3.png  ← Coloca tu tercera imagen aquí
   ```

3. **Una vez agregadas las imágenes:**
   - Los usuarios podrán seleccionar uno de estos avatares predeterminados al registrarse
   - Si no seleccionan un avatar, se usará uno de estos por defecto

## Nota:
Si no agregas estas imágenes, la aplicación seguirá funcionando, pero los usuarios solo podrán subir sus propias imágenes o usar la cámara.
